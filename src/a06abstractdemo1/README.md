抽象类：
不能实例化；
抽象类不一定有抽象方法，但是抽象方法一定是抽象类
abstract 所修饰的方法就是抽象类
子类继承后必须重写抽象方法，否则报错无法运行

接口：行为的抽象
部分类需要使用的外部方法

接口的定义：public interface 接口名{}

接口不能实例化
接口与类：通过 implements 关键字来表示

接口的成员变量：只能是常量，默认使用 public static final
构造方法：无
成员方法：默认使用 public abstract
jkd7 之前：只能定义抽象方法
jkd8：可以定义有方法体的方法(默认 default，静态 static)
jkD9：可以定义私有方法


中间类：适配器类，间接继承
