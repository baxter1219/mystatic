package a05;

{
/\*
_ 包就是文件夹，用于管理不同的 java 类
_ 命名：公司域名的反写+包的作用
_ 例如：com.baodu.domain + 类名（全类名）
_ 导包的情况：
_ 使用本包的类是时候，不用导包
_ 使用 java.lang 包中的类时，不需要导包
_ 其他情况需要导包
_ 使用两个包中的同名类，需要使用全类名
\*/

    /*
     * final:
     * 修饰方法：表示方法是最终的方法，不能重写
     * 修饰类：不能被继承
     * 修饰变量：叫做常量，不能被再被赋值（只能赋值一次）
     */

     常量的命名：
     单个单词：全部大写
     多个单词：全部大写，中间用下划线（_）隔开

     final:修饰的引用数据类型：记录的地址不改变，但是内容是可以修改的


     权限修饰符：
     用来控制成员能被访问的范围
     可以修饰:成员变量，方法，构造方法，内部类

     private:同一个类
     空着不写：同一个类       同一个包中的其他类
     protected:同一个类       同一个包中的其他类   不同包下的子类
     public:所有


    局部代码块：
    {

    }//这个就是局部代码块，用完即回收

    1.构造代码块：优先于方法加载,成员位置的代码块
    2.作用：可以把多个构造方法中的重复代码抽取出来
    3.执行时机：创建本类对象的时候会执行构造代码块，再执行构造方法
}
